1) Iniciar el proyecto nodeJS
2) crear carpeta src para añadir los códigos
3) crear el .babelrc un transpilador de código antiguo a los actuales
4) se crea el webpack.config.js luego se pueden utilizar generadores de proyectos 

5) dentro de src se crea routes para las urls
6) se crea carpeta de models
7) se crea la carpeta public para mandar al navegador con todos los archivos
8) se crea la carpeta app para el cliente con los archivos .vue y lo dedicado al frontend, luego se convierte y se envía a la carpeta public
9)se crea el archivo index.js que está en src y se inicia el servidor con express
10) se fija el puerto con process.env.PORT se pide un puerto el S.O
